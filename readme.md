# 3PPY

A browser-based audio visualizer. Drop in an MP3, watch it come alive.

---

## Features

- Drag & drop or browse to upload any audio file
- Real-time 3D visualization synced to the audio
- Playback progress slider
- Built with Three.js

---

## Getting Started

Requires [Node.js](https://nodejs.org).

```bash
git clone <repo-url>
cd 3ppy
npm install
npm run dev
```

Then open the local URL shown in your terminal.

---

## Usage

1. Drop an MP3 onto the drop zone, or click **browse files**
2. Press **Launch**
3. Hit `Enter` at any time to trigger a pulse effect
