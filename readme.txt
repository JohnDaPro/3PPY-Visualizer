# 3PPY — Audio Visualizer

3PPY is a browser-based audio visualizer built with modern web technologies. It allows users to upload an audio file (such as an MP3) and experience a real-time visual representation of the music using dynamic graphics.

---

## FEATURES

• Drag & drop or browse to upload audio files
• Real-time audio visualization
• Interactive launch screen UI
• Progress slider for playback control
• Clean and minimal design
• Built using Three.js for 3D rendering

---

## HOW IT WORKS

1. Open the application in your browser
2. Drag and drop an MP3 file into the drop zone
   OR click "browse files" to select one manually
3. Once a file is loaded, press the "launch" button
4. The visualizer will start rendering graphics synced to the audio

---

## PROJECT STRUCTURE

index.html     → Main HTML layout
main.css       → Styling and UI design
main.js        → Core logic and visualization code

---

## INSTALLATION

Make sure you have Node.js installed.

1. Clone the repository

2. Install dependencies:

   npm install

3. Run the development server:

   npm run dev

4. Open the provided local URL in your browser

---

## DEPENDENCIES

• Three.js — used for rendering 3D graphics
• Vite (or similar bundler) — for development and build process

---

## NOTES

⚠ This application contains flashing visuals and may not be suitable for users with photosensitive epilepsy.
